Projects

Pedestrian Crossing Intent Prediction from Dashcam Video

While many recent approaches to pedestrian crossing intention rely on large, end-to-end deep models trained on dedicated intent datasets, this prototype explores a lightweight alternative: YOLOv8 + BYTETrack tracking, ego-motion–corrected geometric features, and heuristic fusion (with optional pose and road-segmentation cues) to produce interpretable SAFE / APPROACHING / CROSSING states on dashcam video.

Code (GitHub) Demo video

Surrogate Safety Analysis at Urban Intersection (YOLOv8 + BoT-SORT)

Developed a YOLOv8 + BoT-SORT pipeline to detect and track road users at signalized intersections and compute surrogate safety metrics such as Time-To-Collision (TTC) and Post-Encroachment Time (PET) for near-miss classification across custom intersection zones.

Code (GitHub) Demo video

YOLOv8-L Based Traffic Detection for Dhaka Streets

Trained a custom YOLOv8-L model for Dhaka traffic detection with eight classes (car, bus, auto-rickshaw, rickshaw, motorcycle, truck, van, pedestrian). Achieved strong performance on frequent classes (car, bus, auto-rickshaw).

Code (GitHub) Demo video

Exam Seat Plan Generator & Web App (Deployed at Uttara University)

Developed an automated exam seating system for Uttara University that parses raw exam and student data, applies adjacency and restriction rules, and generates seat plans and PDFs for thousands of students each session, all accessible through a simple Flask web app.

Code (GitHub) Live / Demo

MNIST Neural Network from Scratch

Built a neural network from scratch using only NumPy to recognize handwritten digits. Implemented forward propagation, backpropagation, and gradient descent manually.

Code (GitHub)

Mars Site Selector — Topography Screening Tools

Developed tools to evaluate Mars Orbiter Laser Altimeter (MOLA) digital elevation models for potential landing sites: scanning and scoring low-relief zones, visualizing candidates on color maps, and generating elevation profiles using NumPy, OpenCV, Matplotlib, and Tkinter

Code (GitHub)

Bayesian Search & Rescue — Probabilistic Simulator

Created an interactive Bayesian search simulator that models sequential area searches on a map, applying probabilistic updates based on forecast and realized effectiveness and running Monte Carlo simulations (up to 10,000 iterations) to compare search strategies, using Python with NumPy and OpenCV.

Code (GitHub)

Housing Price Modeling — Regression Practice

Built and evaluated linear regression models for housing prices on a large sales dataset, with feature engineering (location, interactions, time effects) and model validation (cross-validation, RMSE, R²) in Python (scikit-learn, statsmodels).

Code (GitHub)

Earthwork Cut–Fill Calculator — Highway Earthworks

Designed a command-line tool to compute earthwork volumes using average-height, trapezoidal, and prismoidal methods, with support for multi-segment grades, unit selection, and basic data visualization, developed in MATLAB.

Code (GitHub)
