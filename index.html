Flagship Projects

Selected work in transportation, ML, and computer vision

Pedestrian Crossing Intent Prediction from Dashcam Video

Built a dashcam-based pipeline that detects and tracks pedestrians using YOLOv8 and BYTETrack, corrects for ego-motion, and estimates crossing intent over time. Each trajectory is classified into SAFE, APPROACHING, or CROSSING based on motion relative to lanes and curbs, with intent labels overlaid on the video.

Code (GitHub) Demo video

Technologies: Python, YOLOv8, BYTETrack, OpenCV, NumPy, pandas

Surrogate Safety Analysis at Urban Intersection (YOLOv8 + BoT-SORT)

Developed a video-based safety analysis pipeline for signalized intersections using YOLOv8 and BoT-SORT to detect and track road users. The system computes surrogate safety metrics such as Time-to-Collision (TTC) and Post-Encroachment Time (PET), identifies near-miss events, and categorizes them by conflict type and location for proactive safety assessment.

Code (GitHub) Demo video

Technologies: Python, YOLOv8, BoT-SORT, OpenCV, NumPy, pandas

YOLOv8-L Based Traffic Detection for Dhaka Streets

Trained a custom YOLOv8-L model on Dhaka street imagery to detect eight traffic classes (car, bus, auto-rickshaw, rickshaw, motorcycle, truck, van, pedestrian). Focused on robust detection under mixed traffic and occlusion, and analysed performance differences between frequent and rare classes to support downstream traffic analytics applications.

Code (GitHub) Demo video

Technologies: Python, YOLOv8-L, OpenCV, Computer Vision

Exam Seat Plan Generator & Web App (Deployed at Uttara University)

Designed and deployed an automated exam seat planning system that parses raw exam and student data, applies adjacency and restriction rules, and generates seat plans and related documents as PDFs. A Flask-based web interface enables departmental staff to upload inputs and download one-click ZIP packages for thousands of students per session.

Code (GitHub) Live / demo

Technologies: Python, pandas, pdfplumber, FPDF, Flask, HTML/CSS
